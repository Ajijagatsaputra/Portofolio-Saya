Portfolio

Project Yang Dikerjakan

Dibawah ini beberapa project yang sudah saya kerjakan. Dengan menggunakan beberapa teknologi yang populer saat ini seperti Laravel, Flask, React Vite, dan Javascript.

Catalogue Restaurant

Aplikasi Catalogue Restaurant ini dibuat dengan bahasa pemrograman Javaascript. Dengan mengintegrasikan API yang tersedia, aplikasi ini ditujukan guna user dapat melihat daftar resto yang populer, dan dapat juga menambahkan atau menyukai resto yang diinginkan.

HealingGo (Travel App)

Aplikasi HealingGo adalah sebuah aplikasi sistem informasi tentang travel wisata. Aplikasi ini ditujukan kepada seluruh masyarakat khusus nya masyarakat Indonesia, dengan adanya aplikasi HealingGo ini maka dapat membantu masyarakat Indonesia mencari destinasi wisata di wilayah negara Indonesia.

Movie-Catalogue

Movie Catalogue adalah sebuah aplikasi berbasis web dengan menampilkan beberapa daftar film, baik dari yang paling populer maupun yang banyak disukai user. Aplikasi ini dikembangkan menggunakan bahasa pemrograman javascript.

Sistem Rekomendasi Pertanian

Sistem Rekomendasi Pertanian menggunakaan sistem pendukung keputusan metode SAW (Simple Additive Weight). Aplikasi ini dikembangkan dengan Laravel, dengan adanya aplikasi ini pengguna dapat melakukan perhitungan untuk pertaniannya hanya dengan memperhitungkan beberapa kriteria dan bobotnya.

EyeU

EyeU adalah sebuah aplikasi sistem deteksi kesehatan mata normal dan katarak yang berbasis website. Aplikasi ini dikembangkan dengan teknologi AI untuk mendeteksinya dan menggunakan framework Flask.
